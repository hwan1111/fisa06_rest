# 데이터 활용 DB 적재, 분석 및 시각화 진행 (데이터를 입력받고, 우리의 DB에 적재하고 해당 데이터에 적합한 방식으로 시각화한다)

## 프로젝트 개요

이 프로젝트는 **Streamlit**을 이용하여 데이터를 입력받고 이를 **DB**에 적재한 후, 적합한 방식으로 시각화하여 최종 사용자에게 제공합니다. 주어진 데이터셋을 사용하여 웹 애플리케이션을 구현하고, 분석된 데이터를 시각화하여 인사이트를 제공합니다.

**핵심 목표:**
- 데이터 수집
- DB 적재 (Google Sheets 또는 MySQL 사용)
- 데이터 분석 및 시각화
- Streamlit 애플리케이션 구현 (input과 output을 포함)
- 사용자와 상호작용할 수 있는 웹 애플리케이션 제공

---

## 기술 스택

- **Streamlit**: 웹 애플리케이션의 인터페이스 구성
- **Pandas**: 데이터 분석 및 가공
- **MySQL**: 데이터 저장 및 관리
- **Folium**: 지리적 데이터 시각화
- **Plotly**: 인터랙티브 차트 시각화
- **Geopy**: 주소를 위도, 경도로 변환
- **st-gsheets-connection**: Google Sheets와의 데이터 연동

---

## 프로젝트 구성 요소

1. **데이터 입력 및 처리**
   - 사용자로부터 입력을 받기 위한 Streamlit 폼과 사이드바 제공
   - 데이터를 Google Sheets나 MySQL에 저장
   - 입력 받은 데이터를 Pandas로 분석 및 처리

2. **시각화**
   - **Folium**을 사용하여 데이터를 지도에 시각화
   - **Plotly**를 이용하여 카테고리별 평균 별점 등을 시각화하는 차트 제공

3. **DB 연동**
   - Google Sheets에서 데이터를 불러와 DB에 저장
   - MySQL과의 연동을 통해 데이터를 안전하게 관리

4. **Streamlit Application**
   - 사용자 인터페이스(UI) 제공
   - 입력된 데이터를 시각적으로 표현
   - 실시간 데이터 업데이트 기능

---

## 데이터셋

프로젝트에서 사용된 데이터는 다음과 같습니다:

1. **기존 데이터셋**: 수업 중 제공된 예시 데이터
   - 예시 데이터셋을 기반으로 DB에 데이터를 적재하고 분석을 수행

2. **팀에서 선정한 데이터셋**: 프로젝트 수행에 적합하다고 판단한 데이터
   - 주어진 데이터셋을 분석하여 통찰력 있는 정보를 도출

---

## 시연회 내용

1. **데이터 선정 배경**:
   - **Google Sheets**에서 식당 정보를 수집하여, 사용자가 직접 정보를 입력하고 리뷰를 작성할 수 있는 시스템을 구축했습니다.
   - **MySQL**을 활용하여 대규모 데이터 처리 및 안정적인 데이터 관리를 구현했습니다.

2. **분석 기준**:
   - 사용자의 리뷰 데이터를 기반으로 평균 별점을 계산하고, 카테고리별 인기 식당을 분석했습니다.
   - 특정 지역 내의 인기 식당을 지도 상에서 시각화하여, 사용자들이 쉽게 찾아볼 수 있도록 했습니다.

3. **팀원 기여**:
   - **팀원 A**: Streamlit 애플리케이션 구현 및 데이터 입력 폼 구성
   - **팀원 B**: MySQL 데이터베이스 연동 및 데이터 분석
   - **팀원 C**: Folium을 이용한 지도 시각화 및 Plotly 차트 구현

4. **트러블슈팅**:
   - **문제 1**: MySQL 데이터베이스 연결 오류
     - 해결: `.env` 파일에 있는 데이터베이스 자격 증명을 확인하고 수정하여 연결 문제 해결.
   - **문제 2**: 주소 데이터 오류로 인한 지도 표시 오류
     - 해결: **Geopy**를 사용하여 부정확한 주소를 처리하고 지도에 정상적으로 표시되도록 개선.

## 실행 방법

1. 이 프로젝트를 로컬에서 실행하려면, 먼저 `requirements.txt`에 명시된 모든 패키지를 설치하세요:

    ```bash
    pip install -r requirements.txt
    ```

2. Streamlit 애플리케이션을 실행하려면, 아래 명령어를 입력하세요:

    ```bash
    streamlit run main.py
    ```

3. 애플리케이션이 로컬 서버에서 실행되면, 웹 브라우저에서 `http://localhost:8501` 주소로 접속하여 확인할 수 있습니다.
